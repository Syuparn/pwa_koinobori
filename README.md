# pwa_coinobori
鯉のぼり🎏のPWAゲームアプリ https://Syuparn.github.io/pwa_koinobori/

## Abstract
(このwebアプリはスマートフォン向けです。)
画面をタップしてこいのぼりを積み上げよう！

↑のwebページをダウンロードすれば、
擬似アプリとしてオフラインでも遊べます。

- Webpack 4.6.3 (for compiling multiple .ts files)
